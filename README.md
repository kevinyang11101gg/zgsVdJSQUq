# Python学生信息管理系统

## 核心功能
- 登录
- 注册
- 学生管理
- 成绩管理
- 班级管理
- 课程管理
- 宿舍管理
- 任务管理
- 资料管理
- 意见反馈
- 系统管理

## 适用场景
- 教育机构学生信息管理
- 学校学生信息登记与查询
- 教学资源管理与分配

## 技术特点
- 使用Python Flask框架进行后端开发
- 前端采用HTML、CSS、JavaScript等技术
- 数据库使用MySQL
- 提供完整源码和运行文档
- 支持Python3.0以上和MySQL8.0环境
- 使用Vscode和Navicat等工具进行开发和管理
